### Directions:

Take an array of objects listing names and return the names in a formatted string.

### Examples:

```javascript
list([ {name: 'Bart'}, {name: 'Lisa'}, {name: 'Maggie'} ]);
//returns 'Bart, Lisa & Maggie'

list([ {name: 'Bart'}, {name: 'Lisa'} ]);
// returns 'Bart & Lisa'

list([ {name: 'Bart'} ]);
// returns 'Bart'

list([]);
// returns ''
```
